================================================================================
HOW TO RUN THE IN-PERSON REVIEW SYSTEM PROJECT
================================================================================

This file contains step-by-step commands to successfully run the project.

PREREQUISITES:
--------------
1. Python 3.10 or higher installed
2. Node.js 18+ and npm installed
3. API Keys:
   - OpenAI API key (for scoring)
   - Soniox API key (for transcription)

================================================================================
STEP 1: BACKEND SETUP (Python/FastAPI)
================================================================================

1.1. Navigate to the project root directory:
     cd C:\Users\Dharani\Desktop\aXtr\InPerson-ReviewSys

1.2. Create a Python virtual environment:
     python -m venv .venv

     OR (if you have uv installed, recommended for faster installation):
     uv venv --python 3.11

1.3. Activate the virtual environment:
     .venv\Scripts\activate

1.4. Install Python dependencies:
     pip install -r requirements.txt

     OR (if you have uv installed, recommended for faster installation):
     uv pip install -r requirements.txt

================================================================================
STEP 2: ENVIRONMENT CONFIGURATION
================================================================================

2.1. Create a .env file in the project root directory (C:\Users\Dharani\Desktop\aXtr\InPerson-ReviewSys)

2.2. Add the following content to the .env file:

     # Required API Keys
     OPENAI_API_KEY=your_openai_api_key_here
     SONIOX_API_KEY=your_soniox_api_key_here

     # Optional - Soniox Configuration
     SONIOX_API_BASE_URL=https://api.soniox.com
     SONIOX_MODEL_ID=stt-async-v3

     # Optional - OpenAI Configuration
     OPENAI_MODEL=gpt-4o-mini

     Note: Replace the placeholder values with your actual API keys.

================================================================================
STEP 3: FRONTEND SETUP (React)
================================================================================

3.1. Navigate to the ui directory:
     cd ui

3.2. Install Node.js dependencies:
     npm install

3.3. (Optional) Create ui/.env file for API configuration:
     REACT_APP_API_BASE=http://localhost:8000

3.4. Return to project root:
     cd ..

================================================================================
STEP 4: RUN THE APPLICATION
================================================================================

4.1. Start the Backend Server:
     --------------------------
     Make sure you're in the project root directory and the virtual environment is activated.

     Option A - Using uvicorn directly (venv activated):
     uvicorn api.main:app --reload --port 8000

     Option B - Using uv (no activation needed):
     uv run uvicorn api.main:app --reload --port 8000

     The backend API will be available at: http://localhost:8000
     API documentation: http://localhost:8000/docs

4.2. Start the Frontend (in a NEW terminal window):
     ------------------------------------------------
     Open a new terminal/command prompt window.

     Navigate to the ui directory:
     cd C:\Users\Dharani\Desktop\aXtr\InPerson-ReviewSys\ui

     Start the React development server:
     npm start

     The frontend will automatically open in your browser at: http://localhost:3000

================================================================================
STEP 5: VERIFY THE SETUP
================================================================================

5.1. Check Backend:
     - Open browser and go to: http://localhost:8000
     - You should see API information
     - Check API docs at: http://localhost:8000/docs

5.2. Check Frontend:
     - Open browser and go to: http://localhost:3000
     - You should see the React application interface

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Module not found" errors
Solution: Make sure virtual environment is activated and dependencies are installed
          Run: .venv\Scripts\activate
          Then: pip install -r requirements.txt

Issue: "Port already in use" error
Solution: Either stop the process using the port, or change the port:
          For backend: uvicorn api.main:app --reload --port 8001
          For frontend: Set PORT=3001 in ui/.env or use: set PORT=3001 && npm start

Issue: API keys not working
Solution: Check your .env file is in the project root and contains correct keys
          Verify keys are valid and have proper permissions

Issue: Frontend can't connect to backend
Solution: Ensure backend is running on port 8000
          Check CORS settings in api/main.py
          Verify REACT_APP_API_BASE in ui/.env matches backend URL

Issue: npm install fails
Solution: Clear npm cache: npm cache clean --force
          Delete node_modules and package-lock.json, then reinstall
          Make sure Node.js version is 18 or higher
